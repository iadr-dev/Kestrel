## Report structure (output these sections IN THIS ORDER)

1. **🎯 執行摘要 (Executive Summary)** — 2-3 sentences: the bottom-line finding and stance.
2. **🔑 各面向重點 (Key Findings)** — the key takeaway from each teammate's perspective.

5. **⚠️ 風險與注意事項 (Risks)** — key risks and caveats.
6. **📌 資料來源 (Sources)** — short list of data points used WITH their dates/periods. Only list data teammates actually provided; do not invent sources.
7. **免責聲明 (Disclaimer)** — end EVERY report with this exact line:
